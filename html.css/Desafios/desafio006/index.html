Tags em HTML

Em HTML 5 podemos inserir os elementos da lista a seguir. Passeo o mouse sobre os termos para descobrir qual Tag deverá ser utilizada em cada caso.

  • Parágrafos;
  • Títulos;
  • Imagens;
  • Abreveações
  • Código-fontes;
  • Listas numeradas;
  • Listas com demarcadores
  • Lista de definições;
  • Citações; e
  • Links.
